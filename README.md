Salud a Tiempo es una aplicación web progresiva orientada a la organización personal de la salud. Permite registrar citas médicas, llevar seguimiento básico de indicadores como peso y presión, y controlar tratamientos. La app guarda primero la información de forma local y posteriormente puede sincronizarla con una API y una base de datos central, lo que la convierte en una solución accesible, práctica y útil incluso en contextos de conectividad limitada.
